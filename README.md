# Modern Endpoint Automation

A collection of enterprise-safe scripts for managing and automating modern Windows endpoint environments.

## Scope
This repository contains PowerShell scripts commonly used in:
- Microsoft Endpoint Configuration Manager (SCCM / MECM)
- Microsoft Intune
- Co-managed environments
- Enterprise endpoint automation scenarios

## Structure
- PowerShell/Citrix – Citrix Workspace management scripts
- PowerShell/SCCM – Configuration Manager utilities
- PowerShell/Intune – Intune remediation and automation scripts
- PowerShell/Utilities – General endpoint tooling

## Goals
- Share real-world automation patterns
- Provide reusable, enterprise-safe scripts
- Document practical solutions to common endpoint management problems

## Notes
All scripts are sanitized for public sharing and intended for educational and operational use.
